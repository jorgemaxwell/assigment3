This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web

HTML5 is the latest version of the Hypertext Markup Language (HTML) used for creating web pages and applications. It is designed to make it easier to create and use multimedia content on the web, and includes new features and improvements to make web development more efficient and effective. Some of the key features of HTML5 include better support for multimedia content, improved accessibility, and the ability to create complex applications using a variety of tools and programming languages. HTML5 is supported by all major web browsers and has become the standard for web development, making it an essential tool for anyone who wants to create modern, engaging web content
